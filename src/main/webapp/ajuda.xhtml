Ajuda

Para acessar as páginas de cadastro de pacientes e visualização dos dados dos pacienter é necessário realizar o login, informando o nome de usuário e senha. Caso não possua, faça um cadastro e aguarde a liberação do login. Essa liberação ocorrerá após a permissão do administrador do sistema. Não é possível ecessar nenhum item do sistema sem o cadastro prévio.

Caso tenha alguma outra dúvida, envie um email para pacheco.comp@gmail.com e aguarde retorno.
