Bem vindo

No menu ao lado você encontra as opções disponíveis e implementadas no sistema. Abaixo você encontra uma breve descrição de cada uma das opções:

  • Home: é está página que está no momento. Ela aparece sempre que você efeturar login no sistema.
  • Cadastrar pacientes: para cadastrar um novo paciente acesse essa funcionalidade. Nela será necessário preencher o formulário
    com todos os dados dos pacientes e do diagnóstico clínico, bem como uma ou mais imagens da lesão.
  • Visualizar pacientes: para visualizar os pacientes cadastrados no banco de dados acesse essa funcionalidade. Aqui você tem acesso
    ao diagnóstico clínico e as imagens cadastradas.
  • Análise de dados: esta funcionalidade realiza uma análise de dados em relação aos pacientes cadastrados no banco de dados.
    Exemplo de informações que podem ser obtidas pela funcionalidade: quantidade de pessoas atendidas por região, porcentagem de mulheres e homens
    afetados por cada tipo de câncer, tamanho médio das lesões, etc.
  • Configurações: esta função é destinada para alteração de senha, nome de usuário e/ou e-mail.
