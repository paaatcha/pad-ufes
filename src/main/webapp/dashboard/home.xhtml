Bem vindo

No menu da barra superior você encontra as opções disponíveis e implementadas neste sistema. Existem dois tipos de usuários do sistema. Um usuário comum, que é capaz de cadastrar e visualizar pacientes e outro administrador, que é possui permissão de edição, tanto de usuários comuns quanto de paciente. Se em algum momento você carregar uma página e ela lhe retornar que seu usuário não possui permissão para tal funcionalidade, você não será capaz de executá-la. Cada uma das funcionalidades individualmente.

Home: é está página que está no momento. Ela aparece sempre que você efeturar login no sistema.

Cadastrar pacientes: para cadastrar um novo paciente acesse essa funcionalidade. Nela será necessário preencher o formulário com todos os dados dos pacientes. As informações em relação a lesão e inserção de imagens, devem ser realizados pelo aplicativo.

Visualizar paciente: esta funcionalidade deve ser usada para visualizar todos os dados de um único paciente. Para isso é necessário inserir o cartão do sus do paciente para uma busca no banco de dados. Usuários ADMIN poderão editar pacientes já cadastrados.

Visualizar todos pacientes: Todos os pacientes são exibidos em uma tabela. Todavia, nem todos os dados são apresentados. Para uma visualização completa é necessário buscar o paciente na seção anterior. Tenha em mente que o processo pode levar alguns segundos dependendo da quantidade de pacientes a serem recuperados no banco de dados.

Análise de dados: esta funcionalidade realiza uma análise de dados em relação aos pacientes cadastrados no banco de dados. Exemplo de informações que podem ser obtidas pela funcionalidade: quantidade de pessoas atendidas por região, porcentagem de mulheres e homens afetados por cada tipo de câncer, tamanho médio das lesões, etc. Só está disponível para usuários ADMIN.

Configurações: esta função é destinada para alteração de senha, nome de usuário e/ou e-mail

Gerenciar usuários: área de gerenciamento dos usuários do sistema. Aqui você pode habilitar um usuário para utilizar o site bem como conceder ou remover poderes de ADMIN dos demais usuários. Obviamente, só está disponível para usuários ADMINs.

Fichas antigas: para inserir um paciente antigo, ou seja, uma ficha que é anterior a este sistema, acesse ao cadastro de pacientes antigos A diferença entre o cadastro do menu e este cadastro é que as informações de lesões e imagens são inseridas todas em um mesmo formulário sem a necessidade do uso do aplicativo no smartphone.

Pacientes gerais: todos os pacientes cadastrados via aplicativo dos monitores e que não foram para sala de cirurgia são chamados de pacientes gerais. Eles podem ser visualizados em visualizar pacientes gerais

Resolução de problemas: Qualquer problema encontrado, bug ou dúvida em relação ao sistema envie um e-mail para pacheco.comp@gmail.com. O seu contato é importanto para encontrarmos possíveis bugs e corrigi-los para que o sistema fique cada vez mais estável e confiável.
