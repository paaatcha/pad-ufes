Bem vindo

No menu acima você encontra as opções disponíveis e implementadas no sistema. Abaixo você encontra uma breve descrição de cada uma das opções:

  • Home: é está página que está no momento. Ela aparece sempre que você efeturar login no sistema.
  • Cadastrar pacientes: para cadastrar um novo paciente acesse essa funcionalidade. Nela será necessário preencher o formulário
    com todos os dados dos pacientes. As informações em relação a lesão e inserção de imagens, devem ser realizados pelo aplicativo.
  • Visualizar paciente: esta funcionalidade deve ser usada para visualizar todos os dados de um único paciente.
    Para isso é necessário inserir o cartão do sus do paciente para uma busca no banco de dados
  • Visualizar todos pacientes: Todos os pacientes são exibidos em uma tabela. Todavia, nem todos os dados são apresentados.
    Para uma visualização completa é necessário buscar o paciente na seção anterior. Tenha em mente que o processo pode levar alguns segundos
    dependendo da quantidade de pacientes a serem recuperados no banco de dados.
  • Análise de dados: esta funcionalidade realiza uma análise de dados em relação aos pacientes cadastrados no banco de dados.
    Exemplo de informações que podem ser obtidas pela funcionalidade: quantidade de pessoas atendidas por região, porcentagem de mulheres e homens
    afetados por cada tipo de câncer, tamanho médio das lesões, etc.
  • Configurações: esta função é destinada para alteração de senha, nome de usuário e/ou e-mail.

Inserindo um paciente antigo

Para inserir um paciente antigo, ou seja, uma ficha que é anterior a este sistema, acesse ao cadastro de pacientes antigos
A diferença entre o cadastro do menu e este cadastro é que as informações de lesões e imagens são inseridas todas em um
mesmo formulário sem a necessidade do uso do aplicativo no smartphone.
